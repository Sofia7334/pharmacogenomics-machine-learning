# pharmacogenomics-machine-learning
A machine learning model trained on ClinPGx annotations data. It is intended to learn to classify pharmacogenomic drug-response phenotypes from gene–drug interaction data. This is a high school students' passion project experimental model, not a clinical decision tool; it is not meant to predict real patient outcomes.
